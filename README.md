# LinearAlgebra-Assignments

線形代数の授業で取り組んだ課題をまとめたリポジトリです。  
回転行列を使って画像を変換し、数式とプログラムの対応を確認することを目的に制作しました。

## 概要

この課題では、画像の各ピクセル座標に対して回転行列を適用し、画像を回転させる処理を Python で実装しています。

- `draft.py`: 初期実装。45度回転で崩れる例を確認するためのコード
- `sample.py`: 回転の中心を考慮して改善した実装
- `slide.html`: 課題発表用のスライド
- `input.jpeg`: 入力画像
- `output_draft.png`, `output_45.png`, `output_90.png`: 実行結果のサンプル

## 使用技術

- Python
- NumPy
- Pillow
- HTML / Tailwind CSS

## 実行方法

必要なライブラリをインストールします。

```bash
pip install numpy pillow
```

初期実装を試す場合:

```bash
python draft.py
```

改善版を試す場合:

```bash
python sample.py
```

どちらのスクリプトも実行結果は `output.png` として保存されます。  
比較用の画像は、このリポジトリ内で別名でも保存しています。

## 学んだこと

- 回転行列をそのまま実装するだけでは、意図した画像変換にならないことがある
- 回転の中心をそろえるために、平行移動と回転を組み合わせて考える必要がある
- 数式上の行列演算が、プログラムでは座標計算として具体化されることを確認できた

## 補足

`slide.html` では、この課題の背景、失敗例、改善点、結果をスライド形式でまとめています。  
授業内の発表資料として作成したものです。
